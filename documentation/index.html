HTML Structure General layout and content structure

Bolt is a modern and clean creative portfolio which is mobile and retina display ready. There are 5 main sections I want to summarize here. There is the navigational bar, the featured image section, the services section, the meet the team section and the blog.

Navigational Bar

The navigation is made up of a Logo, a tagline and dropdown icons. Each a tag with class icon is an icon that links to a dropdown menu.

To change links in the menu, change the li tags in ul tag with menu class. I have premade two dropdowns, the heart icon which signifies social media, and the menu icon which signifies other pages.

Services

The services section consists of rows of services. Each service consists of a title and a description and in image that will slide in from the left or right. Alternating between left and right is a good way to vary things but keep them uniform.

You can change the title of each service in the h1 tag, and also change the description text in the p tag. The img with class slide-in is the image describing your service that will be animated in as you scroll down.

  <div class="row">
    <div class="col-md-5 col-sm-5 col-xs-12">
      <h1>Responsive.</h1>
      <p>Bolt is responsive meaning it looks great on all devices. It automatically adjusts to the browser size so everything is readable on the smallest mobile phone to the largest desktop computer.</p>
    </div>
    <div class="col-md-7 col-sm-7 col-xs-12">
      <img class="slide-in right" src="img/bolt-all-devices.png" alt="bolt all devices">

Elements HTML Elements

Bolt supports many cool customized bootstrap elements. I will go over elements including the Bootstrap Column system, forms, Collapsible toggle menus, alert boxes, buttons and Modal image previews.

Bootstrap Columns

Bootstrap columns are the easiest way to make things responsive. just define a row div, and you can add columns to them. with tags like col-sm-4, that means you can fit 3 of those into a column on a small device. There are 12 columns in a row. Usually you want to define col-xs-12 as 12 columns because on small screens things will shrink.

  <div class="row">
  <div class="col-md-4 col-sm-4 col-xs-12">
</div>

JavaScript Files

Tag Description

Masonry Blog

Masonry is a plugin that allows things in a row to fit nicely together without unnecessary whitespace.

Here is the initializing code. If you wanna customize any masonry options it would go inside the $container.masonry({ block.

  var $container = $('.masonry-grid');
  
  $container.imagesLoaded(function(){
    $container.masonry({
      itemSelector : '.item',
      gutter : 0
    });
    $(".item").css("opacity", "1");
  });
